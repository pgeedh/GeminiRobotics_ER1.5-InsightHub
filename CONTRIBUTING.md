## ü§ù How to Contribute

### 1. Adding a Use Case
Found a new way to use Gemini ER 1.5? (e.g., "Safety Checking" or "Text Reading").
1.  Create a script in `examples/`.
2.  Add a generic image or GIF to `assets/` (or use a public link) showing the result.
3.  Submit a PR with the tag `[USE-CASE]`.

### 2. Improving Documentation
See a typo? Want to clarify a prompt pattern?
- PRs for `README.md` and `RESOURCES.md` are highly encouraged.
- Please stick to the "Professional & Clean" aesthetic.

### 3. Reporting Issues
- If a script doesn't work with the latest API version, please open an Issue tagged `[BUG]`.
- For model-specific feedback (hallucinations, refusal), please use the official Google AI Studio feedback tools.
